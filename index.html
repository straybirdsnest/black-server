这是一个实验项目

项目组里的 iR$k 认为 TA 的 GitHub 账号下一直没有一个像样的项目，所以决定借由这个项目来做一个实验，初期只是使用 Spring 框架做一个简单的，拥有用户账户管理功能的服务器，同时还带有 OAuth 第三方认证的账户登录功能。在账户管理成形之后，考虑在加入在线聊天的功能。

Spring

主流的轻量级 Java EE 开源框架，本项目中使用 Spring-Security 来完成用户登录验证的功能，使用 Spring-Security-Oauth 来完成第三方账户登录的功能。整体配合Spring MVC 实现一个简单的 Web 应用。
